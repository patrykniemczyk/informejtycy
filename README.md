# Informejtycy - programming learning platform 🚀

Welcome to our free online platform designed to help you learn programming! With over 1,500 users already benefiting from our interactive lessons and exercises, we're excited to offer a comprehensive and intuitive environment for learning programming concepts and honing your skills.

Visit us at [informejtycy.pl](https://informejtycy.pl).

## Technologies Used 💻

Our platform is built using a combination of modern technologies to provide both a smooth user experience and efficient backend performance.

### Frontend 🌐
- **HTML**: Defines the structure of the pages.
- **CSS**: Handles the design, layout, and responsiveness of the user interface.
- **JavaScript**: Manages dynamic behavior such as form submission, view updates, and communication with the backend via the `fetch` API.
- **Hugo**: A static site generator used to build the site from Markdown (`.md`) documents, enabling fast page generation and easy management of content.

### Backend ⚙️
- **Python (Flask)**: Our backend framework. Flask handles routing, application logic, and managing the task database. It also handles user-submitted solutions and runs tests to validate the correctness of the code.
- **Sandbox Environment**: Code submitted by users is compiled and run in a secure sandbox environment where it is tested against predefined test cases. If the code passes all tests, it is considered correct.
- **HTTPS Communication**: The frontend communicates with the backend using direct HTTPS requests through Flask endpoints, ensuring efficient and secure data transfer.

### Server Configuration 🖥️
- **VPS Hosting**: Our application is hosted on a Virtual Private Server (VPS) for scalability and reliability.
- **Nginx**: Acts as a reverse proxy, forwarding HTTPS requests to the Flask application running behind Gunicorn.
- **SSL Certificates**: We use Let’s Encrypt to secure communication between users and the website via SSL/TLS encryption.
